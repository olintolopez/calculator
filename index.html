MathCalc1.0
  1. Press ✕ button, ESC of your keyboard, or click outside of the modal to close it
  2. You can use the mouse to select a number, followed by an operator, then another number and finally you select the equal key to get the result.
  3. If you get the ERROR MATH result on the screen where the result is displayed, you must use the Clr key to erase everything and start a new calculation.
  4. If you want to calculate a series of basic operations you can (as indicated above) select the digit followed by an operator, select the other digit and to continue the sequence select the other operator and then the next digit and so on at the end of the sequence select the equal key.
  5. You can use the keyboard of your pc to evaluate a sequence of basic operations. Next we have
